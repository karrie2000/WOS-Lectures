Intro to Flexbox

The Flexbox System has two sets of properties.

  1. Flex container
  2. Flex items

When we assign an element's display property to flex, we turn that element into a flex container and all direct children become flex items.

"The parent tells the children how to behave."

Flex items are only as wide as they need to be (by default).

Flexbox is a one-dimensional system.

Important Flex Container Properties:

  • display: flex;
  • flex-direction: row (default) || column
  • justify-content:
    • Governs items on the main axis (x-axis or row by default)
    • flex-start (default)
    • flex-end
    • center
    • space-between
  • align-items:
    • Governs items on the cross axis (y-axis or column by default)
    • stretch (default)
    • flex-start
    • flex-end
    • center
  • gap: 20px (for example)

Important Flex Item Properties:

  • align-self:
    • Allows a flex item to misbehave.
    • stretch
    • flex-start
    • flex-end
    • center
  • flex:
    • A shorthand property that sets more than one property at once.
    • Defines how many pieces of the pie a child is allowed to eat.
    • Its value is a unit-less number.
